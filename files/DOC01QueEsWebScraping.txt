Web Scraping
____________
Web scraping Es una técnica usada por data scientist y backend developers para extraer información de internet, 
accede a esto usando el protocolo de tranferencias de hipertexto (HTTP) o a través de un navegador.
Los datos extraídos usualmente son guardados en una base de datos, incluso en una hoja de cálculo 
para posteriores análisis. Puede hacerse de manera automática (bot) o manualmente.

Que es Xpath
____________
Es un lenguaje que sirve para apuntar a las partes de un documento XML.
Xpath modela un documento XML como un árbol de nodos. 
Existen diferentes tipos de nodos: elementos, atributos, texto.

En últimas, el web scraping sirve para extraer datos de sitios web, los cuales podemos
transformar en información para nuestras aplicaciones...
